# 微推 WeiTweet

A chrome extension written in Vue & Vuex. Post Weibo and Twitter in one click.

-

一键编辑发送推特与微博。
Post Weibo and Twitter in one click.

-

- 统一编辑、一键分享
- 网页图片自动抓取
- 支持本地图片、图片链接

-

- Multi-editing
- One click to send them all
- Auto extracts images from page.
- Image upload. Url pasting.

-

开源安全，不收集任何数据，不占后台。
